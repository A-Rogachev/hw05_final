Используемые технологии

  • Python
  • Django

Данное веб-приложение создано с помощью фреймворка Django 2.2.9, написанного на языке программирования Python. Для оформления использовался Bootstrap - популярный инструмент для создания адаптивного дизайна сайтов.

На данный момент уже реализовано:
  • Возможность регистрации и авторизации пользователя
  • Создание и редактирование публикаций
  • Просмотр публикаций в зависимости от группы поста
  • Просмотр детальной информации о публикации

В дальнейшем функциональность веб-приложения будет расширена (я надеюсь), будут добавлена возможность подписки на авторов и др.
